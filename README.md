# Inventory & Sales System
A web-based Inventory & Sales System designed to help businesses manage products, monitor sales, and streamline operations. The project is *currently under development*, with core features such as product management, dashboard overview, and user authentication already implemented.

## Features
- [x] **Dashboard Tab**: Provides a quick overview of system metrics and key performance indicators.
- **Products Tab**:
  - [x] View products with essential details
  - [x] Edit product information
  - [x] Create new products
  - [x] Delete existing products
  - [x] Search and filter functionality
- [ ] **POS Tab**: TBD Point of Sales
- [ ] **Customer List**: A list of customers and their past sales history
- **Authentication:**
  - [x] User login and logout
  - [x] Basic session management
  - [ ] 'Remember Me' feature
- **Sales Tab:**
  - [ ] Record new sales transactions
  - [ ] Track sales history and generate reports
- **Inventory Management:**
  - [ ] Automatic stock adjustments based on sales and purchases
  - [ ] Low-stock alerts
- **User Management:**
  - [x] Role-based access control
  - [ ] Manage multiple users
- **Reports & Analytics**:
  - [x] Visual charts and insights for products, sales, and inventory
- **Exporting Data**:
  - [ ] CSV or PDF import for sales and inventory reports
 
## Technical Stack
- **Frontend**: React.js, Tailwind CSS (v4)
- **Backend**: Laravel 12
- **Database**: SQLite
- **Authentication**: Laravel Sanctum
